## 渲染模式

* 静态生成
在构建时生成 HTML 的预呈现方法。然后，预呈现的 HTML 将在每个请求中重复使用。

* 服务器端渲染
服务器端呈现是在每个请求上生成 HTML 的预呈现方法。

* getStaticProps
仅在服务器端运行, 永远不会在客户端运行。甚至不会包含在浏览器的JS包中。这意味着可以编写诸如直接数据库查询之类的代码，而无需将它们发送到浏览器。
