# 页面加载时
主要监控分析资源加载、首屏渲染耗时等

## 资源加载
主要是js、css文件的加载, 还有图片、音频等相关资源。

## 页面性能
主要是spa、ssr等首屏渲染相关耗时。

包含以下阶段, DNS -> TCP -> SSL -> 请求响应 -> 内容传输 -> DOM解析 -> 资源加载

## 接口
统计 接口请求总数、失败次数、成功次数; 计算 接口成功率、平均耗时、接口请求阶段耗时; 以及4xx、5xx等分量。

# 页面运行时
主要是错误上报、和错误分析。
